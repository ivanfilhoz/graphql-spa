# graphql-spa

This is an example app featuring the following tools:

- React
- Vite
- Tailwind
- shadcn/ui
- gql.tada
- i18next

See the live demo [here](https://graphql-spa.netlify.app/).

## Running locally

Clone this repository and install the dependencies with:
```
$ npm install
```

Then, run the app in development mode:
```
$ npm run dev
```
